# Текстовый Квест: Таинственный Остров

## Описание проекта
Интерактивная текстовая новелла, в которой игрок исследует загадочный остров, принимая решения, влияющие на сюжет и финал истории. Проект реализован на языке C# и работает в консоли.

## Особенности
- **15 сцен** с различными вариантами развития событий
- **5 уникальных концовок**, зависящих от выбора игрока
- **Несколько путей прохождения** - разные комбинации выборов ведут к разным финалам
- **Интуитивное управление** - выбор осуществляется вводом номера варианта

## Установка и запуск

### Требования
- .NET SDK версии 6.0 или выше
- Любая операционная система с поддержкой .NET

### Запуск проекта
1. Скопируйте код в файл `Program.cs`
2. Откройте терминал в папке с файлом
3. Выполните команды:
```bash
dotnet new console -n InteractiveNovel
dotnet run
